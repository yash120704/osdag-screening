# Osdag Screening Task

This repository contains the solution for the Osdag screening task involving
2D and 3D visualization of shear force and bending moment diagrams.

## Contents
- Task-1: 2D SFD and BMD for central longitudinal girder
- Task-2: 3D SFD and BMD for all girders using node coordinates and element connectivity

## Tools Used
- Python
- Xarray
- Matplotlib

## How to Run
```bash
python src/task1_2d_plots.py
python src/task2_3d_plots.py
